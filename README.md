# Book-shop-flask
 **Описание**:  

Сайт магазина книг.

 **Как запустить проект?**: 
 
  Установить все зависимости
  
 ```
 pip install -r requirements.txt
```

и прописать команду:
 ```
python app.py
```
Далее перейти по локальному адресу. В Flask это обычно http://localhost:5000 или http://127.0.0.1:5000. 

Есть на хостинге: https://book-shop-flask.onrender.com

## Основные возможности
- Каталог книг с фильтрацией по категориям, цене, новинкам/бестселлерам/акциям
- Корзина, оформление заказа, личный кабинет
- Админ-панель: управление книгами, категориями, пользователями, заказами
- Чат (общие комнаты + личные сообщения) с API на Flask
- Отзывы и рейтинг книг
- Загрузка изображений книг и категорий

## Технологии
- Python 3, Flask, SQLAlchemy
- Bootstrap 5, Jinja2
- SQLite / PostgreSQL
- REST API для чата
